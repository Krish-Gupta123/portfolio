# Krish Gupta — Portfolio Website

A polished, responsive personal portfolio built with HTML, CSS, and JavaScript. This site highlights experience, achievements, and project work with modern interaction design.

## Overview

This repository contains a static portfolio website designed to present a developer profile, showcase completed projects, and provide easy contact options. The site is optimized for desktop and mobile, with animated sections and clean visual styling.

## Key Features

- Responsive layout for desktop, tablet, and mobile
- Animated hero section with typed title effects
- Custom cursor and scroll-based entrance animations
- Project showcase cards with technology tags
- Achievement cards with certificate preview support
- Contact section with email, GitHub, LeetCode, and phone links

## Project Structure

```text
portfolio/
├── index.html              # Main page markup
├── css/
│   └── style.css           # Site styling and layout
├── js/
│   └── script.js           # Interactive behavior and animations
└── images/                 # Image assets and certificates
    └── dsa course completion certificate.jpg
```

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, custom properties)
- JavaScript (ES6+)
- GSAP for motion and scroll-triggered animation
- Typed.js for typed text effects
- Google Fonts for typography

## Local Preview

To preview locally, open `index.html` in any modern web browser.

For a local development server, use one of these options:

- VS Code Live Server extension
- Python:
  ```powershell
  python -m http.server 8000
  ```
- Node.js (if installed):
  ```powershell
  npx http-server .
  ```

Then visit `http://localhost:8000`.
